# 🤖 AI Engineering Learning Tracker

> Daily habit tracker for my journey to becoming an AI Engineer.

Every morning a GitHub Action opens a new issue with:
- Today's study topic
- Specific tasks to complete
- A resource link
- A prompt to log my progress

I reply to each issue to log what I did. This keeps me accountable and builds my GitHub contribution graph daily.

## 🔧 How it works

A GitHub Actions workflow runs every morning at **8:00 AM IST** and opens a new issue with the day's study plan. I comment on it with my progress update.

Each completed issue = a contribution on my graph = visible proof of consistency.
